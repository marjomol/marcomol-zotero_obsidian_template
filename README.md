# Zotero → Obsidian Advanced Template

A structured template to integrate Zotero annotations into Obsidian, designed for deep reading, synthesis, and knowledge building.

## ✨ Features

- Automatic import of Zotero metadata and annotations
- Structured sections for:
  - Summary
  - Connections
  - Questions
  - Definitions
- Integrated Obsidian → Anki-ready fields
- Visual thinking sections (mindmaps, schemes) to be filled with mermaid.
- Annotation categorization by color

## 🧠 Philosophy

This template is designed to follow a 3-step learning process:

1. **Capture** → Highlights & annotations from Zotero  
2. **Process** → Comments, connections, and structure  
3. **Synthesize** → Personal summaries and insights

## ⚙️ Requirements

- Obsidian  
- Zotero  
- Zotero Integration plugin  
- Nunjucks templating (used internally by the plugin)  

Optional:
- Obsidian to Anki plugin (for spaced repetition workflow)

## ⚡ Quick Start

1. Install Zotero Integration plugin in Obsidian  
2. Add `template.md` to your templates folder  
3. Select it inside the plugin settings  
4. Import any Zotero item  

→ You should immediately get a fully structured note
